para controlar onde o site de destino vai abrir, podemos usar o atributo target que suporta os seguintes valores:

  • _blank vai abrir o link em uma nova janela em branco
  • _self vai abrir o link na janela ou frame atual (padrão)
  • _top vai desfazer todos os frames e abrir o destino no navegador completo
  • _parent similar ao uso _top em referência à janela mãe
  • nome-do-frame caso esteja usando frames, indicar o nome da janela abrir
